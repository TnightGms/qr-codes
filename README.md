# ✨ QR Studio Pro
![Versión](https://img.shields.io/badge/Versión-1.0.0-blue)
![Licencia](https://img.shields.io/badge/Licencia-MIT-green)
![Autor](https://img.shields.io/badge/Autor-tnightgms-orange)

Una aplicación web moderna para generar códigos QR totalmente personalizables. Diseñada con una interfaz **Glassmorphism** y fondos dinámicos. Creado por **tnightgms**.

## 🚀 Características
* **Generación instantánea:** Transforma cualquier enlace, texto o red social en un QR en milisegundos.
* **Personalización total:** Elige el color exacto de los píxeles del QR y su color de fondo.
* **Descarga en 1 clic:** Guarda el código QR directamente en tu dispositivo en formato PNG.
* **Diseño Premium:** Interfaz responsiva con efecto cristal y animaciones fluidas CSS.

## 🛠️ Tecnologías utilizadas
* **HTML5** (Estructura semántica)
* **CSS3** (Glassmorphism, Gradientes animados, Flexbox)
* **JavaScript ES6** (Fetch API, manipulación del DOM)
* **API Externa:** [goQR.me API](http://goqr.me/api/)

## 🌐 Cómo usarlo
El proyecto está desplegado a través de GitHub Pages. No necesitas instalar nada.
1. Entra a la web del proyecto.
2. Escribe tu texto o pega tu enlace URL.
3. Selecciona tus colores favoritos en la paleta.
4. Haz clic en **Generar**.
5. ¡Descárgalo y úsalo donde quieras!
